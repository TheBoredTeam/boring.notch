Pi · brand palettes · bundled manifest

Curated brand palettes,
lifted out of Swift.

Today the curated overrides for mono / off-palette logos live in a hardcoded static dict — brandColorOverrides in PiAgentManager.swift, two entries. Adding a brand means editing Swift and cutting a release. Move that table into a bundled BrandPalettes.json, decoded once at init into the exact same [String:[NSColor]]. Same lookup, same memory, no new dependency — and adding a brand is a one-line JSON edit. Thirteen new curated palettes below, every stop anchored to the toolkit's real logo fill — toggle any one with vs without the override to see why it's needed.

why a manifest, not a db

Three stores, three natures

The earlier question was sqlite-vs-memory. The honest split: only one of these is curated source-of-truth — and that's the only one worth persisting as data you can edit. The caches stay exactly where they are.

cache · regenerable

Logo bytes

NSCache + per-slug disk PNG. A cold miss costs one CDN fetch, then it's warm. A DB wouldn't cut a single network call the disk cache doesn't already.

stays in memory + disk →
derived · ~ms to recompute

Palette memo

[String:[NSColor]] derived from the SVG bytes. Persisting it is caching a cache. Losing it on restart is a non-event.

stays in memory →
curated · can't regenerate

Brand overrides

Hand-authored gradient stops for mono / off-palette logos. This is the one that wants to live as editable data — so a new brand doesn't need a recompile.

Pick a toolkit — see the aurora it earns

Each stop is grounded in the actual logos.composio.dev/api/<slug> fill (shown as the source swatch). The toggle flips between the two real code paths: with override publishes the curated stops raw — hand-picked to read on black; without override is what the notch derives on its own — the SVG/raster palette, gated and legibleTint(0.45)'d. For true mono marks (GitHub, Sentry, Browserbase) that auto path has no usable hue to work with, so it collapses to the generic indigo or a flat pastel wash — which is exactly the gap the curated override fills. the bundled file

BrandPalettes.json

Ships in the app bundle. version lets the loader stay forward-compatible; note documents curator intent so the table doesn't rot into mystery hex. Keys are the lowercased toolkit slug the sidecar emits as event.toolkit.

the swift change

    
A decoded table replaces the literal

    
Add two Decodable structs and a one-shot loader. Everything downstream — the derivePalette lookup, the saturation-gate bypass for curated stops — stays byte-for-byte identical, because it still reads a [String:[NSColor]] keyed by slug.

    

      
 // PiAgentManager.swift

      
-private static let brandColorOverrides: [String: [NSColor]] = [

      
-    "notion": [hex("#C9C7C1"), hex("#8C8A85")],

      
-    "github": [hex("#8B949E"), hex("#6E7681")],   // only two, hardcoded

      
-]

      
+private struct BrandPalette: Decodable { let stops: [String]; let note: String? }

      
+private struct BrandManifest: Decodable { let version: Int; let palettes: [String: BrandPalette] }

      
+

      
+// decoded once at init; identical shape to the old literal

      
+private static let brandColorOverrides: [String: [NSColor]] = loadBrandPalettes()

      
+

      
+private static func loadBrandPalettes() -> [String: [NSColor]] {

      
+    guard let url = Bundle.main.url(forResource: "BrandPalettes", withExtension: "json"),

      
+          let data = try? Data(contentsOf: url),

      
+          let m = try? JSONDecoder().decode(BrandManifest.self, from: data)

      
+    else { return [:] }   // missing/corrupt file → behave as today (no override)

      
+    return m.palettes.reduce(into: [:]) { acc, kv in

      
+        acc[kv.key.lowercased()] = kv.value.stops.compactMap(NSImage.colorFromHex)

      
+    }

      
+}

      
 // derivePalette(slug:…) — curated branch publishes stops RAW (no legibleTint):

      
-    publishPalette(override.map(Self.legibleTint))   // washed the brand to pastel

      
+    publishPalette(override)                          // trust the curator — raw

      
 // raster/histogram fallback path KEEPS legibleTint — colors there are unpredictable
