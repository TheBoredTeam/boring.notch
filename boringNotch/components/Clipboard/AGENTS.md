# AGENTS.md

## Purpose

- This subtree owns clipboard history capture, storage, menu-bar UI, and related view-model behavior.

## Ownership

- Applies to every file under `boringNotch/components/Clipboard`.
- Parent instructions from the repository root and `boringNotch/components` remain binding.

## Local Contracts

- Clipboard history is opt-in and must not record content until enabled.
- Skip transient, concealed, and autogenerated pasteboard types.
- Store image payloads locally under Application Support and bound retention by item count and storage size.
- Clipboard history persistence may run off the main actor, but pending saves must be flushed before app termination or destructive store clears.

## Work Guidance

- Keep pasteboard polling lightweight and avoid recapturing writes initiated by clipboard history actions.
- Preserve real image data when copying image history items back to the pasteboard.
- Cache derived list state, source icons, and thumbnail-sized image previews instead of doing disk or workspace lookups from row rendering.

## Verification

- Build the macOS app target after changing this subtree.
- Manually verify text copy, image copy, copy-back, deletion, clearing, and relaunch persistence when UI behavior changes.
